API Endpoints

Below are described the REST endpoints available that you can use to filter animals

All

                    http://http://www.animaliarestapi.com/all

Name

Search by animal name.

                    http://http://www.animaliarestapi.com/name/{name}
                
                    http://http://www.animaliarestapi.com/name/Aardwolf

Alphabetical

Search by animal's first character.

                    http://http://www.animaliarestapi.com/alpha/{character}
                
                    http://http://www.animaliarestapi.com/alpha/a

Phylum

Search by animal's Phylum

                    http://http://www.animaliarestapi.com/phylum/{phylum}
                
                    http://http://www.animaliarestapi.com/phylum/Chordata

class

Search by animal's class

                    http://http://www.animaliarestapi.com/class/{class}
                
                    http://http://www.animaliarestapi.com/class/Mammalia

Order

Search by animal's order

                    http://http://www.animaliarestapi.com/order/{order}
                
                    http://http://www.animaliarestapi.com/Order/Carnivora

Family

Search by animal's family.

                    http://http://www.animaliarestapi.com/family/{family}
                
                    http://http://www.animaliarestapi.com/family/Hyaenidae

Genus

Search by animal's genus.

                    http://http://www.animaliarestapi.com/genus/{genus}
                
                    http://http://www.animaliarestapi.com/genus/Proteles

species

Search by animal's species.

                    http://http://www.animaliarestapi.com/species/{species}
                
                    http://http://www.animaliarestapi.com/species/Proteles cristatus

Location

Search by location.

                    http://http://www.animaliarestapi.com/location/{location}
                
                    http://http://www.animaliarestapi.com/location/Southern Africa

Diet

Search by the foods.

                    http://http://www.animaliarestapi.com/diet/{food}
                
                    http://http://www.animaliarestapi.com/diet/ants

Response Example

Filter Response

You can filter the output of your request to include only the specified fields.

                    http://http://www.animaliarestapi.com/{service}?fields={field},{field},{field}
                
                    http://http://www.animaliarestapi.com/all?name,class
